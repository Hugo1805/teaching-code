# teaching-code

teaching-code es un repositorio educativo donde documento conceptos fundamentales de programación, lógica computacional y estructuras básicas de código. Incluye ejemplos, pseudocódigo, ejercicios prácticos y buenas prácticas en JavaScript y TypeScript. Ideal para quienes están comenzando su camino como desarrolladores o para enseñar a otros de forma estructurada y progresiva.
